# 백준 11652번 - 카드
[문제 링크](https://www.acmicpc.net/problem/11652)
## 문제
민규는 숫자 카드 N장을 가지고 있다. 숫자 카드에는 정수가 하나씩 적혀있다. 민규는 가장 많이 가지고 있는 숫자 카드에 적힌 수를 알아내려고 한다. 가장 많이 가지고 있는 숫자 카드가 여러 가지라면, 그 중에서 가장 작은 수를 알아내려고 한다.
## 내가 푼 아이디어
```python
from collections import defaultdict
N = int(input())
dic = defaultdict(int)
for _ in range(N):
    num = int(input())
    dic[num] += 1
max_k = float('inf')
max_v = 0
for k, v in dic.items():
    if v > max_v:
        max_k = k
        max_v = v
    if v == max_v and k < max_k:
        max_k = k
print(max_k)
```
## 풀이 설명
1. 입력받은 숫자 카드들을 딕셔너리에 저장하여 각 숫자가 몇 장 있는지 카운트한다.
2. 딕셔너리를 순회하면서 가장 많이 가지고 있는 숫자 카드의 수와 그 숫자를 갱신한다. 만약 같은 수의 카드가 여러 개 있다면, 가장 작은 숫자를 선택한다.
3. 최종적으로 가장 많이 가지고 있는 숫자 카드에 적힌 수를 출력한다.
