# 백준 10814번 - 나이순 정렬
[문제 링크](https://www.acmicpc.net/problem/10814)
## 문제

```python
n = int(input())
def set_list(my_lst):
    dict = {}
    dict['age'] = int(my_lst[0])
    dict['name'] = my_lst[1]
    return dict
total_lst = []
for i in range(n):
    str = input()
    total_lst.append(set_list(str.split()))
sort_dict = sorted(total_lst, key=lambda i : i['age'])
for i in sort_dict:
    print(i['age'], end=' ')
    print(i['name'])
```
## 풀이 설명
1. 입력받은 나이와 이름을 딕셔너리 형태로 저장하는 `set_list` 함수를 정의한다.
2. `total_lst` 리스트에 입력받은 나이와 이름을 딕셔너리 형태로 저장한다.
3. `sorted` 함수를 사용하여 `total_lst`를 나이 기준으로 정렬한다.
4. 정렬된 리스트를 순회하면서 나이와 이름을 출력한다.
