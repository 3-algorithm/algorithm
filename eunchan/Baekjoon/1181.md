# 백준 1181번 - 단어정렬
[문제 링크](https://www.acmicpc.net/problem/1181)
## 문제
알파벳 소문자로 이루어진 N개의 단어가 주어졌을 때, 다음 조건에 따라 단어를 정렬하는 프로그램을 작성하시오.
1. 길이가 짧은 것부터 정렬한다.
2. 길이가 같으면 사전 순으로 정렬한다.

## 내가 푼 아이디어
```c
#include <stdio.h>
#include <string.h>
#include <stdlib.h>
typedef struct _word_t{
    char w[50];
    int l;
}word_t;
int compare(const void* a, const void* b){
    word_t* p_word = (word_t*)a;
    word_t* q_word = (word_t*)b;
    if(p_word->l != q_word->l){
        return p_word->l - q_word->l;
    } else {
        return strcmp(p_word->w, q_word->w);
    }
}
int main(){
    int n = 0, max = 0;
    scanf("%d",&n);
    word_t word[n];
    word_t* pword = word;
    memset(word,0, sizeof(word));
    for(int i=0;i<n;i++){
        scanf("%s",word[i].w);
        word[i].l = strlen(word[i].w);
    }
    qsort(pword,n,sizeof(word_t),compare);
    printf("%s\n",word[0].w);
    for (int i=1;i<n;i++) {
        if (!strcmp(word[i].w, word[i-1].w)) continue;
        printf("%s\n",word[i].w);
    }
    return 0;
}
```
## 풀이 설명
1. 단어를 저장하기 위한 구조체 `word_t`를 정의한다. 이 구조체는 단어를 저장하는 배열 `w`와 단어의 길이를 저장하는 정수 `l`로 구성된다.
2. `compare` 함수는 `qsort`에서 사용할 비교 함수로, 단어의 길이를 먼저 비교하고, 길이가 같으면 사전 순으로 비교한다.
3. `main` 함수에서는 입력받은 단어들을 `word_t` 구조체 배열에 저장하고, 각 단어의 길이를 계산하여 저장한다.
4. `qsort` 함수를 사용하여 단어들을 정렬한다.
5. 정렬된 단어들을 출력하는데, 중복된 단어는 한 번만 출력한다. 이를 위해 이전 단어와 현재 단어를 비교하여 중복을 제거한다.
