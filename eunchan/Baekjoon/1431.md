# 백준 1431번 - 시리얼 번호
[문제 링크](https://www.acmicpc.net/problem/1431)
## 문제
시리얼 번호는 대문자와 숫자로 이루어진 문자열이다. 시리얼 번호를 정렬하는 방법은 다음과 같다.
1. 길이가 짧은 시리얼 번호가 먼저 온다. 예를 들어, 길이가 2인 시리얼 번호는 길이가 3인 시리얼 번호보다 먼저 온다.
2. 길이가 같으면, 시리얼 번호에 포함된 숫자의 합이 작은 시리얼 번호가 먼저 온다. 예를 들어, "ABCD"는 숫자가 없으므로 합이 0이고, "A1B2"는 숫자가 1과 2이므로 합이 3이다. 따라서 "ABCD"는 "A1B2"보다 먼저 온다.
3. 길이도 같고, 시리얼 번호에 포함된 숫자의 합도 같으면, 사전 순으로 앞서는 시리얼 번호가 먼저 온다. 예를 들어, "AAA"는 "AAB"보다 사전 순으로 앞서므로 "AAA"는 "AAB"보다 먼저 온다.
## 내가 푼 아이디어

```python
N = int(input())

lst = []
for _ in range(N):
    lst.append(input())
lst.sort(key=len)
# print(lst)
for i in range(N-1):
    for j in range(0, N-1-i):
        if len(lst[j]) == len(lst[j+1]):
            suma = sumb = 0
            for a in lst[j]:
                if a.isdecimal():
                    suma += int(a)
            for a in lst[j+1]:
                if a.isdecimal():
                    sumb += int(a)
            if sumb < suma:
                lst[j+1], lst[j] = lst[j], lst[j+1]
            elif suma == sumb:
                for a in range(len(lst[j])):
                    if lst[j+1] < lst[j]:
                        lst[j+1], lst[j] = lst[j], lst[j+1]
for i in lst:
    print(i)
```
## 풀이 설명
1. 입력받은 시리얼 번호들을 리스트에 저장한 후 길이순으로 정렬한다.
2. 길이가 같은 시리얼 번호들에 대해서는 숫자의 합을 계산하여 비교한다. 숫자의 합이 작은 시리얼 번호가 먼저 오도록 정렬한다.
3. 길이도 같고 숫자의 합도 같은 시리얼 번호들에 대해서는 사전 순으로 정렬한다.
4. 최종적으로 정렬된 시리얼 번호들을 출력한다.
